# Hims & Hers Health (NYSE: HIMS) — Equity Analyst Dashboard

A single-page equity analyst dashboard for Hims & Hers Health, designed for institutional / sophisticated retail investors doing event-driven equity research.

## View it

Open `index.html` in any browser, or view the rendered version via GitHub Pages (if enabled in repo settings).

## What's inside

- **Equity snapshot** — price, multiples, key positioning at a glance
- **FY2025 revenue trajectory** — modelled revenue path with quarterly milestones
- **Recent acquisitions (Q2 2024 → today)** — M&A activity and integration thesis
- **Catalysts strengthening the bull case** — near-term and structural drivers
- **Tone shift across the last four earnings calls** — sentiment analysis of management commentary
- **Refresh tone for the next earnings call** — forward-looking thematic guide

## How it was built

Generated from a custom equity-research workflow built on the WAT framework (Workflows / Agents / Tools):
- Data sources: Bloomberg / yfinance fundamentals, earnings call transcripts, M&A press releases, FDA / regulatory filings
- Tone analysis: NLP scoring across Q1 2024 – Q1 2025 earnings calls
- Rendered as a self-contained HTML artifact (no build step, no dependencies) — printable, shareable, archivable

## Design

Calm, clinical, considered. Inspired by the warm cream / sage / charcoal palette of the Hims & Hers brand — wellness restraint, not trading-terminal aggression. Designed for focused desktop reading sessions, not at-a-glance dashboards.
